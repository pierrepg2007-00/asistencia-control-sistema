# control_academico_web

Proyecto web sencillo para gestionar información académica usando Python, HTML, CSS, JavaScript básico y archivos JSON.

## Objetivo

Organizar el control académico de estudiantes, materias, matrículas, notas, asistencia y reportes mediante una estructura simple para trabajo en equipo.

## Módulos

1. Estudiantes.
2. Materias y periodos.
3. Matrículas.
4. Notas.
5. Asistencia.
6. Reportes.

## Tecnologías usadas

- Python.
- HTML.
- CSS.
- JavaScript básico.
- JSON.

## Estructura general

```text
control_academico_web/
├── data/
├── core/
├── web/
├── static/
├── evidencias/
├── reportes_generados/
└── prompts/
```

## Integrantes y módulos asignados

- Integrante 1: módulo Estudiantes y estructura inicial del proyecto.
- Integrante 2: módulo Materias y periodos.
- Integrante 3: módulo Matrículas.
- Integrante 4: módulos Notas y Asistencia.
- Integrante 5: módulo Reportes.
